AtCorder Beginner Contest 162　でのB問題

FizzBuzz列のN項目までに含まれる数の和を求める問題


            /*
            *　問題：FizzBuzz列のN項目までに含まれる数の和を求めてください。
            *　入力：標準入力から N を与えられる。
            *　出力：FizzBuzz列のN項目までに含まれる数の和を出力せよ。
            *　制約： 1 < N < 10^6
            *　例題：
            *  入力：15　　出力：60
            *  FizzBuzz列の15項目までは次の通りです。
            *  1, 2, Fizz, 4, Buzz, Fizz, 7, 8, Fizz, Buzz, 11, Fizz, 13, 14, FizzBuzz 
            *  15項目までに1,24,7,8,11,13,14が含まれ、これらの和は60です。
            */
            
            #include <iostream>
            using namespace std;
                
            int main(){
            unsigned long int n,i = 0;
            unsigned long int sum = 0;
            cin >> n;
                
            while(n != 0){
                i++;
                if(i % 3 != 0 && i % 5 != 0)
                sum += i;
                n--;  
            }
                
            cout << sum;
            }

AtCorder Beginner Contest 176　でのB問題

整数Nが9の倍数であることと、Nを十進法で表したときの各桁の数の和が9の倍数であることは同値です。

Nが9の倍数であるか判定する問題


            /*
            *　問題：Nが9の倍数であるか判定してください。
            *　入力：標準入力から N を与えられる。
            *　出力：Nが9の倍数ならば Yes、そうでないなら No を出力せよ。
            *　制約： 1 < N < 10^200000　Nは整数
            *　例題：
            *  入力：123456789　　出力：Yes
            *  各桁の数の和は1+2+3+4+5+6+7+8+9=45であり、
            *  45は9の倍数なので、123456789は9の倍数。
            */
            #include <iostream>
            using namespace std;
            
            int main(){
            string n;
            int sum = 0;
            cin >> n;
            
            if(n.size() > 0)
            {
                for(int i = 0; i < n.size(); i++)
                {
                int num = n.at(i) - '0';
                sum += num;
                }
                if(sum % 9 == 0)
                cout << "Yes";
                else
                cout << "No";
            }
            else
                cout << "No";
            }

AtCorder Beginner Contest 176　でのC問題

N人が1列に並んでおり、前からi番目の人の身長はA¡です。

それぞれの人の足元に、高さが0以上の踏み台を設置し、全ての人が次の条件を満たすようにしたいです。

条件：踏み台を込めて身長を比較したとき、自分より前に、自分より背の高い人が存在しない

この条件を満たす時の、踏み台の高さの合計の最小値を求める問題。


            /*
            *　条件：踏み台を込めて身長を比較したとき、自分より前に、自分より背の高い人が存在しない。
            *　問題：前述の条件を満たす時の、踏み台の高さの合計の最小値を求めてください。
            *　入力：標準入力から N　A¡ ... Anを与えられる。
            *　出力：Nが9の倍数ならば Yes、そうでないなら No を出力せよ。
            *　制約： 1 < N < 2×10^5  1 < A¡ < 10^9　入力は全て整数
            *　例題：
            *  入力：123456789　　出力：Yes
            *  各桁の数の和は1+2+3+4+5+6+7+8+9=45であり、
            *  45は9の倍数なので、123456789は9の倍数。
            */
            #include <iostream>
            #include <vector>
            using namespace std;
            
            int main()
            {
            int N;
            cin >> N;
                
            vector height(N);
            for(int i = 0; i < N; i++)
            {
                cin >> height.at(i);
            }
            long long sum = 0;
            
            for(int i = 0; i < N-1; i++)
            {
                if(height.at(i) > height.at(i + 1) )
                {
                sum += height.at(i) - height.at(i+1);
                height.at(i+1) = height.at(i);
                }
            }
            cout << sum;
            }
